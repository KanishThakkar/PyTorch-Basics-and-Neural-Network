Learning repository containing PyTorch fundamentals and implementation of a basic neural network using PyTorch.

This repository contains Jupyter notebooks covering:

1. PyTorch Fundamentals  
   -Random Tensors
   -Matrix Multiplication
   -Matrix Multiplication Rules:
   -Tensor Aggrigation
   -Arg__()
   -Reshaping , stacking , squezzing and unsquezzing Tensors
   -Converting numpy array into pytorch tensor and converting back pytorch Tensor to numpy array
   -Reproducibility (trying to take the random out of random)
   -Putting tensor on gpu

2. Basic Neural Network  
   - Building a simple  neural network
   - Forwardpropogation
   - Loss functions
   - Gradient  
   - Backpropagation  
   - Training loop
